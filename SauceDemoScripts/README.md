### Windows

### macOS / Linux
```bash
chmod +x run.sh
./run.sh
```

## Manual Setup

1. Install [Node.js LTS](https://nodejs.org)
2. `npm install`
3. `npx playwright install --with-deps`
4. `npx playwright test`
5. `npx playwright show-report`

## Requirements

- Node.js 18+
- npm 9+
- (Optional) VS Code with Playwright extension
